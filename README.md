# ☕ POS Cafe System

A complete real-time Point of Sale system for cafes and restaurants.

## Tech Stack
| Layer | Tech |
|---|---|
| Frontend | React + Vite + Zustand + TanStack Query |
| Backend | Node.js + Express |
| Database | PostgreSQL (Knex ORM) |
| Realtime | Socket.io |
| Auth | JWT + Zod |
